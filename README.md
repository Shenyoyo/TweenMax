安裝nodejs 最新穩定版本  
NVM指令  
nvm list：查看已安裝的版本  
nvm list available：查看有哪些Node 版本可以裝  
nvm install v12.16.1：安裝指定的 Node 版本  
nvm use  v12.16.1：指定 Node 版本  
開啟cmd 使用檢查版本   
node -v
